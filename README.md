# AWS Landing Zones — 3 Security Levels

## Overview
This repository contains 3 AWS Landing Zone templates 
built with Terraform, each with a different security level.

## Security Levels

### 🟢 Basic — SMB
- Single account
- Essential security
- Cost optimized

### 🟡 Standard — Mid-Market  
- Multi-account
- RGPD compliant
- Enhanced security

### 🔴 Advanced — Enterprise
- Full multi-account
- ISO 27001 compliant
- Maximum security

## Prerequisites
- Terraform >= 1.0
- AWS CLI configured
- AWS Account
